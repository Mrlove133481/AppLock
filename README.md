#### 简介  
AppLock应用锁，保护你的隐私。       
欢迎Start,Fork和Issues。
项目用到的第三方库：litepal     
在5.0以上的系统需要获取 查看应用使用情况 的权限，否则应用锁会失效   
若有好的建议例如实现方案，界面，功能等欢迎提出。

 

目前问题：
-  进程的保活还没实现好
-  项目中的代码结构逻辑没完善好，还存在一些bug。

附：
在github上看到一个判断应用在前后台的方法，非常牛逼，可以替换项目中原来的方法来实现，推荐一波。
